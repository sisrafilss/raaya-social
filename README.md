# RAAYA SOCIAL

**Deployment Link:** [https://raaya-social-alpha.vercel.app/](https://raaya-social-alpha.vercel.app/)

## Project Overview

This is a responsive, modern landing page for Raaya Social, a social media management agency helping brands and creators scale their short-form video presence. It was developed to showcase service offerings, client portfolio, pricing options, and client testimonials. 

The project utilizes semantic HTML, pure CSS with custom flexbox and grid layouts, and Vanilla JavaScript for interactive elements (such as the responsive mobile navigation). 

This is a job task provided by SM Technology.
